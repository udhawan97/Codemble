The authoritative roadmap lives in the repo's
[CLAUDE.md](https://github.com/udhawan97/Codemble/blob/main/CLAUDE.md) and moves
only when milestones actually complete. The shape:

## NOW — Phase 0: complete v1, Python only

Parser → galaxy (semantic zoom) → grounded explanations → Python language lens
→ checks → illumination + star chart. Everything, one language, end-to-end.

## NEXT — more languages

TypeScript/JavaScript via a tree-sitter adapter, language filter/switch for
polyglot projects, TS/JS idiom lens. Then Go / Rust / Java, plus
level-of-detail rendering for larger repos.

## LATER — reach

A shareable **read-only galaxy link** (the only planned cloud touch), extra
quest types (trace-a-request, fix-the-failing-test), and the coordinated public
launch.

## Non-goals

No accounts. No telemetry. No hosted code. No XP or streaks — illumination and
the star chart are the whole game. No free-flight camera. Ever.
